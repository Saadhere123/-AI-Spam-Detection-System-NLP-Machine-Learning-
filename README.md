# 📩 AI Spam Detection System (NLP + Machine Learning)

A machine learning-based web application that detects whether a message is **Spam or Ham (Not Spam)** using NLP techniques and multiple classification models.

This project is built for learning, internship submission, and portfolio showcasing.

---

## 🚀 Features

- 📊 Spam vs Ham classification
- 🤖 Multiple ML models comparison:
  - Naive Bayes
  - Support Vector Machine (SVM)
  - Logistic Regression
- 🧹 Text preprocessing (cleaning, lowercasing, noise removal)
- 📈 Accuracy comparison using bar chart
- 💾 Best model saved using `best_model.pkl`
- 🌐 Interactive web interface using Streamlit
- 🔍 Real-time prediction system

---

## 🧠 Machine Learning Workflow

1. Data Collection (Spam/Ham messages)
2. Text Preprocessing
3. TF-IDF Vectorization
4. Model Training (3 ML models)
5. Model Evaluation
6. Best Model Selection
7. Model Deployment (Streamlit App)

---

## 📊 Dataset

- Total Samples: **200**
- Classes:
  - `spam` → Promotional / Fraud messages
  - `ham` → Normal messages
